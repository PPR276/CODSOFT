## 📂 Tasks Completed

### ✅ Task 1 – To-Do List Application

A command-line based task management system that allows users to:

- Add tasks
- Remove completed tasks
- View existing tasks
- Clear the entire task list

The application stores tasks in a `tasks.txt` file using file handling, ensuring persistence of data.

**Concepts Used:**
- File Handling (`read`, `write`)
- Lists
- Conditional Statements
- Loops
- User Input Handling

---

### ✅ Task 2 – Basic Calculator

A simple calculator program that performs:

- Addition (+)
- Subtraction (-)
- Multiplication (x)
- Division (/)

The program takes user input and performs the selected arithmetic operation.

**Concepts Used:**
- Conditional Logic (`if-elif`)
- Arithmetic Operators
- User Input Handling

---

### ✅ Task 3 – Random Password Generator

A secure password generator that:

- Generates passwords of user-defined length
- Ensures inclusion of:
  - Special characters
  - Lowercase letters
  - Uppercase letters
  - Numbers
- Uses Python’s `random` module for complexity

**Concepts Used:**
- Random Module
- Lists & Nested Lists
- Loops
- String Concatenation
- Conditional Logic

---

### ✅ Task 4 – Rock Paper Scissors Game

A command-line interactive game where:

- User plays against computer
- Computer randomly selects move
- Score tracking is implemented
- Game continues until user exits

**Concepts Used:**
- Functions
- Global Variables
- Loops
- Conditional Logic
- Random Module
- Score Tracking System
